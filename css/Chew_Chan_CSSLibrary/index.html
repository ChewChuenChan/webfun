Button

When we set up a button,the first step is set the width and height.

Then, set the background-color and border.

We also can set up the thickness, style and color in Border.

circle button can form by setting the border-radius into bigger values in px or %.

button {

width: 200 px;

height: 300 px;

background-color: white;

border: 2 px solid black;

border-radius: 50 px

}

Flex-ible Column

The Flex Shorthand

We can use the flex shorthand property to create columns that can resize themselves.

First, set the parent element.row to display:flex;

Second,set the child element.column to flex:ratio;

For example:when we set flex:1,that means each column in that row has ratio 1 out of the total amount.

If there are 3 columns, then each column occupy 1/3 of the row.

.row {

display:flex;

}

.column {

flex:1;

}

If we set flex:2,that means that column in that row has ratio 2 out of the total amount.

If there are 3 columns, then the flex:2 column occupy 2/4 of the row. While the other 2 columns will share the 1/4 of the total space.

.row {

display:flex;

}

.column1 {

flex:2;

}

.column2 {

flex:1;

}

Navigation Bar

display:flex

Flex can used to position smaller elements (like images or boxes) within a larger element.

Flex displays an element as a block-level flex container

justify-content

The justify-content property aligns the flexible container's items horizontally".

  • justify-content: flex-start;
  • justify-content: flex-end;
  • justify-content: center;
  • justify-content: space-between;
  • justify-content: space-around;
  • justify-content: space-evenly;

align-items

The align-items property aligns the flexible container's items vertically".

  • align-items:flex-start;
  • align-items:flex-end;
  • align-items:center;
  • align-items:baseline;
  • align-items:stretch;

This time we set the commends in parent element .nav to affect child elements.nav-title, .nav-links

We need to set two containers into left & right, so we will use justify-content:space-between

We also need to set them in the middle, so we will use align-items:center

.nav {

display:flex;

justtify-content:space-between;

align-items:center;

}

Overflow

The overflow property specifies whether to clip content or to add scrollbars

when an element's content is too big to fit in a specific area.

  • overflow:visible;
  • overflow:hidden;
  • overflow:clip;
  • overflow:scroll;
  • overflow:auto;

.box {

background-color: #fec5bb;

font-size: 64 px;

border: 4 px solid black;

margin: 20 px;

padding: 0 px 3 px;

width: 223 px;

//we need it has a scroll-bar added, set it as overflow:auto;

overflow: auto;

}
